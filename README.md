# Enchanting Convergence: Reenchanted

A modern port of **Enchanting Convergence** to Minecraft 1.20.1

---

## About

**Enchanting Convergence: Reenchanted** is a small utility mod that overhauls Minecraft’s enchanting system, inspired by the mod *Enchanting Plus*.

Instead of relying on random enchantments, this mod allows players to:
* Select specific enchantments
* Progressively upgrade enchantment levels
* Pay costs in XP, lapis, and enchanting power

This creates a more controlled and strategic enchanting experience.

---

## Status

> This port is considered a **work in progress**.
> Some features may change or be incomplete.

---

## Features

* Deterministic enchanting (no RNG)
* Progressive enchantment leveling
* Custom cost system:
    * XP
    * Lapis Lazuli
    * Enchanting Power (bookshelves)
* Scrollable enchantment list
* Custom enchanting table GUI

## Building

This project is ready to import into IntelliJ IDE with all relevant configurations saved for convenience.

To build the mod without using IDEs:

```bash
./gradlew build
```

The output `.jar` will be located in:

```
build/libs/
```
